
# TrulyLegal Consulting Website

A modern, responsive, interactive landing page for TrulyLegal Consulting and TrulyVirtual.

## Features

- Responsive modern layout
- Animated hero and floating cards
- Light/dark theme toggle
- Mobile menu
- Scroll reveal effects
- Service filtering
- FAQ accordion
- Contact cards and CTA buttons
- GitHub Pages friendly

## How to use

1. Upload the files to a GitHub repository.
2. Open `index.html` in a browser or deploy via GitHub Pages.
3. Replace contact details, links, and copy as needed.
4. Wire the form to your backend or a service like Formspree / Netlify Forms if desired.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Notes

- Prices and rates are intentionally excluded.
- The site is written as a one-page conversion-focused website.
- No framework is required.
